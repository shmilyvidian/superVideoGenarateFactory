# 06 Storyboard Image Generation Prompt

```text
你是顶级抖音电商信息流分镜导演和 AI 生图提示词工程师。

你的任务：基于复刻蓝图、视频画面拆解、仿写文案、产品信息、产品图说明，先写出可供 Codex 内置 GPT image/image2 生图模型使用的分镜图生成提示词，然后必须实际生成分镜图图片。不要把“分镜图提示词”当成最终交付；最终交付必须是实际分镜图图片。

输入信息：
- 视频画面拆解：{{  }}
- 复刻蓝图：{{  }}
- 视频文案脚本仿写：{{  }}
- 产品信息：{{  }}
- 产品图说明：{{  }}
- 产品图附件：{{  }}

核心原则：
1. 复刻蓝图决定分镜图张数、每张分镜图包含哪些关键分镜、每张分镜图对应的原视频时间范围。
2. 视频画面拆解决定每一格的场景、主体、动作、机位、景别、构图、背景、光线和画面事件。
3. 仿写文案只用于判断每一格承载的卖点和脚本推进，不改变原视频镜头结构。
4. 分镜图必须一比一复刻对标视频的真实镜头内容。原视频是唯一的镜头来源；产品图只用于替换产品外观，不能提供新镜头、新场景、新动作、新道具、新机位、新背景或新的产品呈现方式。
5. 产品一致性、不形变是P0门槛，优先级高于画面好看和构图相似。产品只要颜色、形态、长度/比例、厚薄、材质、纹理、结构、Logo/标识位置、Logo/标识大小、Logo/标识颜色、Logo/标识方向不符合产品图，就不能判定为合格。
6. 不得用“减少logo、弱化logo、隐藏logo”来规避AI文字/标识风险。只要原视频对应镜头的拍摄角度应该看到产品logo所在侧面，分镜图就必须保留logo/标识，并且必须在真实物理位置、真实比例、真实方向上出现。
7. 每一格生成前必须判断：这个镜头角度理论上该不该看到logo/标识。如果该看到，就写入明确位置和角度；如果不该看到，只允许因为物理角度、鞋/手遮挡或非logo侧不可见而不出现，不能把logo移到脚背正面、脚底、袜头、包装、鞋子或背景上。
8. 分镜图用于表达对标视频的脚本结构、镜头顺序、关键画面和卖点推进，不是首尾帧，不是直接拿来当视频首帧。
9. 每张分镜图未来会对应一段即梦视频，所以必须严格继承复刻蓝图中的分镜图规划，不能自行把多张合并成一张。
10. 所有分镜图都必须生成9:16竖版画布，不生成1:1方图。
11. 单张分镜图最多12格，同时单张分镜图通常对应原视频时长最多15秒；若对标视频总时长约15-17秒，默认一张分镜图压缩为15秒。
12. 如果复刻蓝图规划第1张为8宫格、对应0:00-0:14，就必须生成第1张8宫格，不得改成12宫格。
13. 如果复刻蓝图规划第2张为6宫格、对应0:14-0:27，就必须生成第2张6宫格，不得合并到第1张。
14. 任何一张分镜图都禁止超过12格；除15-17秒短视频默认压缩例外外，不得覆盖超过15秒的原视频内容。
15. 如果某张不足12格，只按实际关键分镜数量生成，不补无关镜头，不新增空镜头，不为了排版合并关键镜头。
16. 每一格必须对应复刻蓝图中的一个关键分镜，不要新增、删除、替换或打乱顺序。
17. 产品外观以用户上传的产品图为最高优先级，必须严格参考产品图的形态、颜色、材质、Logo位置和可见细节；包装结构、包装颜色、数量、排列方式和使用方式只有在原视频本来存在同类镜头时才允许参考，不能因此新增画面。
18. 不要强行新增原视频没有的任何画面元素，包括新场景、新动作、新道具、新机位、新背景、新产品呈现方式、吊牌、包装文字、卖点文字、字幕、角标。产品卖点主要通过原视频已有画面动作和产品细节体现。
19. 分镜图必须符合真实物理世界运行逻辑：重力方向合理，手和产品真实接触，人物姿势可实现，产品大小比例合理，衣物/头发/液体/链条/包装等材质运动真实，光线和阴影一致，镜头方向和空间关系连续。
20. 不要生成不可能的画面：手指穿过产品、产品悬浮无支撑、链条断裂或反重力、人物关节扭曲、同一产品在相邻格突然变形/变色/变数量、前后场景地理关系矛盾。
21. 如果分镜图里有人物、达人、KOC、口播博主或明显手部/身体角色，第1张分镜图需要锁定人物/手部/服装/背景/光线/手机实拍质感。第2张及后续分镜图必须延续第1张的真实视觉风格；当生图工具支持图片参考时，使用实际生成出来的第1张分镜图作为视觉参考。
22. 如果当前生图工具无法真实附加产品图或第1张分镜图，只能用文字硬约束产品和人物一致性；若生成结果明显漂移，不要把它当最终成品，应重新生成。
23. 如果复刻蓝图规划多张分镜图，必须一次性生成全部分镜图，不要只生成第1张。
24. 生成后保存到 `outputs/storyboard_images/` 或当前任务的最终输出目录。最终交付给用户时展示/链接实际图片文件，不只给提示词。

生图提示词写法：
- 开头写清：9:16竖版、X宫格、布局方式、真实手机实拍质感、不是海报、不是广告主视觉、无字幕无文字无水印。
- 写清产品图最高优先级，明确禁止把原视频产品错误带入新产品。
- 写清产品一致性硬约束，包含颜色、长度/比例、厚薄、袜口/边缘结构、纹理、袜头袜跟/端部形态、脚弓/局部纹理、Logo/标识物理位置、大小、颜色、方向，以及禁止变成的错误产品。
- 每一格写成一段完整生图描述，包含场景、主体、动作、产品替换、机位景别、构图、光线、质感、产品一致性、logo/标识可见性判断、物理动作逻辑。
- 对每一格都要写清该角度是否应该看到logo/标识：应该看到时必须说明位置和方向；不应该看到时说明被角度、鞋或手自然遮挡，不得移动到错误位置。
- 多张分镜图时，每张都要写清对应的原视频时间范围、关键分镜范围和宫格数量。

输出/执行要求：
1. 先内部形成每张分镜图的生图提示词。
2. 立即调用 Codex 内置 GPT image/image2 生图模型生成真实分镜图图片。
3. 生成数量必须等于复刻蓝图规划数量。
4. 生成后必须逐格质检：先看产品一致性和不形变，再看logo/标识物理逻辑，再看真实物理逻辑，再看原视频分镜相似度。
5. 整图生成一次并质检一次；如果只有1-2个关键错格，允许针对性重做一次；如果大面积跑偏或重做后仍有P0产品错误，直接判失败并说明原因，不继续无限重生。
6. 最终不要只输出提示词；最终交付必须有图片文件。
```
