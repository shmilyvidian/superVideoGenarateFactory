你的任务：基于原视频拆解、复刻蓝图、仿写文案、产品信息、产品图说明、产品图附件和已经实际生成出来的分镜图附件，输出可直接复制到即梦 Seedance 2.0 全能参考模式使用的视频生成提示词。

- audio_mode：{{ silent / ambient / music / voiceover / full }}
- 声音补充要求：{{  }}

图片附件输入：

- 分镜图1段：已作为图片内容输入，对应第1段视频。
- 分镜图2段：如复刻蓝图规划多段，则对应第2段视频。
- 分镜图3段：如复刻蓝图规划多段，则对应第3段视频。

核心逻辑：
1. 产品图锁产品：产品品类、外观、颜色、形态、比例、材质、纹理、Logo/标识/文字/花纹位置、大小、颜色、方向，以及吊坠、扣件、宝石、接口、标签、包装、屏幕、按钮、瓶盖、喷头、纹理等本品类可见关键细节，以产品图上传和产品图说明为最高优先级；包装结构、数量、组合方式、排列方式和使用方式只在原视频本来存在同类镜头时才允许参考，不能因此新增镜头内容。
2. 产品一致性是P0门槛。即使分镜图里某格产品有轻微漂移、logo不清楚或局部形变，视频提示词也必须明确拉回产品图；分镜图只能锁构图、镜头顺序、场景、光线和人物/手部风格，不能覆盖产品图的真实外观。
3. 如果分镜图中出现产品品类错误、形态错误、颜色/款式错误、比例不准、材质不准、数量错误、关键结构缺失，或Logo/标识/文字/花纹/吊坠/扣件/宝石/接口/标签/包装等关键识别细节缺失、不清楚、错位，必须在全局设定中明确写成“分镜图误差，禁止继承”。产品外观唯一以产品图为准；最终视频中该出现的关键识别细节不能缺失，也不能移动到错误物理位置。
4. 分镜图锁结构和一致性：分镜图决定每段视频的镜头顺序、构图、场景、光线、人物/手部风格、产品摆位和画面质感。
5. 原视频拆解锁时间码和镜头顺序：每个镜头的时长、动作推进和画面内容必须对应原视频拆解。
6. 仿写文案锁配音节奏：视频镜头切换、人物口型、手势和画面动作要能匹配仿写文案的节奏。
7. 真实物理世界锁动作：所有镜头必须符合重力、空间、比例、接触、反射、阴影、人物姿态、材质运动和镜头连续性。
8. 最终提示词必须用具像化导演语言输出，让没看过原视频的人只看提示词也能想象画面。
9. 一比一复刻锁画面来源：原视频拆解和分镜图是唯一镜头来源。产品图只替换产品外观，不得引入原视频没有的新镜头、新场景、新动作、新道具、新机位、新背景或新的产品呈现方式。

长视频一致性规则：
1. 如果原视频约15-17秒，默认压缩成一张分镜图和一段15秒 Seedance 提示词；如果明显超过17秒，按复刻蓝图拆成多张分镜图、多段 Seedance 提示词，后续在剪映拼接。
2. 前后段一致性只通过产品图和分镜图控制。
3. 第2段及后续段落可以引用当前段分镜图，也可以引用第1张分镜图作为风格/人物/场景连续参考；这仍然属于“通过分镜图控制”。
4. 禁止引用第一段已生成视频、上一段生成视频或任何生成后的视频片段来维持一致性。
5. 不上传对标视频作为 Seedance 参考。原视频只通过拆解、蓝图、仿写文案和分镜图进入提示词。

图片编号规则：
1. 最终即梦生成时，默认先按产品图上传字段里的顺序上传全部产品图。
2. 如果产品图上传字段里有N张产品图，则产品图在即梦里对应@图片1、@图片2、@图片3……直到第N张。
3. 每段生成时，当前段分镜图在产品图之后上传，所以当前段分镜图编号为@图片N+1。
4. 第2段及后续段落如果额外上传第1张分镜图作为统一风格参考，则第1张分镜图编号为@图片N+2。
5. 最终输出中必须把@图片N+1、@图片N+2替换成实际编号。禁止输出@图片X、@图片N、@图片N+1这类占位符。
6. 禁止输出或要求使用任何视频参考编号。

分段规则：
1. 有几张分镜图，就输出几段视频提示词。
2. 分镜图1段对应第1段视频提示词；分镜图2段对应第2段视频提示词；分镜图3段对应第3段视频提示词。
3. 每段时长必须控制在15秒以内。
4. 每段时间码必须根据原视频拆解、复刻蓝图、仿写文案和对应分镜图自然分配，不要机械平均。
5. 每段内部要完整、连贯，但不要强行生成整条超15秒视频。

声音规则：
1. 声音按 audio_mode 执行，不再默认禁止环境音、动作音效、背景音乐或人声口播。
2. silent：静音模式，无环境音、无动作音效、无背景音乐、无人声口播。
3. ambient：保留真实环境音和动作音效，例如脚步声、倒水声、开盖声、冰块声、摇晃声、包装摩擦声、产品接触声和空间氛围声；不要人声口播，不要背景音乐。
4. music：保留环境音和动作音效，并加入适合电商广告节奏的背景音乐；不要人声口播。
5. voiceover：保留环境音和动作音效，并加入人声口播/旁白；背景音乐不生成或只保留极轻铺底。
6. full：环境音、动作音效、背景音乐和人声口播全部允许；人声不能盖过产品关键动作音效，背景音乐不能盖过口播。
7. 除非 audio_mode=silent，否则不要写“无声音、无背景音乐、无环境音”。无论哪种模式，默认仍然禁止屏幕字幕和画面文字。

人物出镜规则：
1. 如果原视频没有人物口播，只生成产品、手部、脚部、身体局部、使用动作和场景，不要强行生成达人正脸。
2. 如果原视频是达人口播、KOC种草或A-roll+B-roll结构，视频生成时必须延续分镜图里的人物形象、性别、年龄段、穿着风格、气质、站位、景别、表情、手势和口播状态。
3. 人物为随机AI达人模特，不参考任何真实人脸，不复刻原视频真人五官。
4. 口播类镜头必须加入对应仿写文案，让人物口型、表情、手势和镜头节奏尽量匹配文案配音节奏。
5. 口播类视频是否允许人物说话、人物配音、背景音乐和环境音，必须按 audio_mode 判断；禁止字幕和画面文字。
6. 非口播产品展示类视频也可以按 audio_mode 保留真实环境音、动作音效和背景音乐；只有 audio_mode=silent 时才写无声音。

镜头描述规则：
每个镜头必须按当前段分镜图的宫格顺序输出。宫格顺序按从上到下、从左到右读取。
每格必须写清楚：
时间码 → 对应第几格 → 景别/运镜 → 主体动作 → 通过什么动作展示什么卖点 → 场景环境 → 构图 → 光线 → 画面质感 → 声音设计（环境音/动作音效/背景音乐/人声口播按 audio_mode 写） → 产品一致性约束 → 关键识别细节/Logo/标识物理逻辑 → 物理世界约束。

禁止模糊表达：
不要只写“展示产品”“突出卖点”“高级感”“氛围感”“参考分镜图”“转化收口”。
必须写清楚：谁在做什么动作，产品在哪里出现，镜头怎么拍，卖点通过什么画面动作体现，场景是什么，光线是什么，画面看起来是什么质感，动作为什么在真实世界里成立。

真实物理世界约束：
1. 手部动作必须能真实完成，手指不能穿过产品，不能凭空拿住无接触物体。
2. 产品大小、重量、悬挂、摆放、拉伸、回弹、倒出、喷洒、佩戴等动作必须符合真实物理规律。
3. 链条、衣物、头发、袜子、包装、液体、纸盒、金属反光等材质运动必须合理。
4. 光源方向、阴影、反射、景深和运动模糊要前后一致。
5. 相邻镜头里的空间关系和产品状态不能突然矛盾，除非有明确剪辑切换。
6. 产品不能在镜头之间无故变形、变色、变数量、变Logo位置或变材质。
7. 高风险连续动作要改成稳定状态或硬切描述。如果提取、穿戴、插入、拉伸、绕挂、打开、数量变化等动作容易造成克隆、穿模、悬浮或肢体错乱，必须写成“硬切到已经完成/已经拿出/已经穿好”的稳定画面，并说明不要表现连续过程。
8. 多件产品镜头必须写清数量关系。如果一件产品从组合中拿出，要么只拍单件，要么明确剩余组合减少一件；不能让同一件产品同时出现在手里和完整组合里。
9. 手脚身体局部镜头必须写清可见肢体数量、接触点和支撑方向，禁止额外手脚、奇怪袖口/裤腿、悬空脚、手指穿过产品。

产品一致性约束：
1. 每格必须明确产品外观以产品图为准，不能继承分镜图中的产品品类错误、形态错误、颜色/款式错误、比例错误、材质错误、数量错误、关键识别细节缺失、模糊、错位或形变。
2. 对有logo/标识/文字/花纹/吊坠/扣件/宝石/接口/标签/包装/屏幕/按钮/瓶盖/喷头/特殊纹理等关键识别细节的产品，每格必须判断该角度是否应显示对应细节。应显示时写清位置、大小、颜色/材质、方向和随产品表面/链条/布料/包装/金属件的透视变化；不应显示时写清是因为角度、手/身体/道具遮挡、背面/非细节面、裁切或景深虚化。
3. 禁止把关键识别细节移动到错误物理位置，禁止强行摆正，禁止变成装饰花纹，禁止放大缩小到不符合产品图比例，禁止为规避生成难度而删除产品标识。若产品图本身没有logo或文字，禁止凭空生成logo、文字或品牌标识。

卖点画面化规则：
1. 透气卖点：用透光、近距离纹理、手指轻撑开纱线等可见动作表达。
2. 柔软卖点：用手掌轻抚、贴合皮肤、自然回弹、面料顺滑等动作表达。
3. 弹力卖点：用双手拉伸、松开回弹、袜口展开等动作表达。
4. 舒适不勒：用袜口贴合脚踝、不勒出压痕、手指轻松伸入边缘等动作表达。
5. 防脱/稳定：用穿鞋、走动、整理后袜口不滑落等动作表达。
6. 价格/规格/组合：不要生成价格文字。只有当原视频本身存在多件组合、包装、平铺、开箱或规格展示镜头时，才允许用相同画面结构表达价格/规格/组合；如果原视频没有这类镜头，只能通过原视频已有收尾镜头或后期字幕表达，不能新加包装、平铺或组合摆拍。
7. 首饰类：用佩戴位置、手指捏起比例、链条自然垂坠、金属/宝石真实反光、衣领搭配、近景翻转细节表达精致和通勤百搭。
8. 其他品类按同样逻辑处理：每个卖点都必须转化为可看见的动作、细节、对比或物理互动。

输出格式：
只输出最终视频生成提示词，不输出分析过程，不输出表格，不输出教程说明，不输出上传素材顺序，不输出即梦参数建议。
