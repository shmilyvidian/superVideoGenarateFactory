# 电商平台档案库（platform-profiles）

chat 版视频 skill 的**平台维度单一事实源**。SKILL 第 0 步据此解析出「本次运行参数」，物化后贯穿全流程。**只读你需要的那个平台节 + 通用部分**（省 token）。

---

## 0. 覆盖优先级（总则）

```
用户运行时明确指定  >  平台档案默认  >  全局默认
```

适用字段：**视频比例、genre、文案语言、时长、字幕开关、audio_mode**。平台档只是「省得每次问」的智能默认，用户任何一项明说都优先。

**全局默认（= 默认档，无平台时用，保证零回归）**：genre=`social-ad`，比例=`9:16`，语言=用户指定/默认中英，选角=欧美女性，字幕=禁，audio_mode=`ambient`。

---

## 1. genre（输出用途，二选一）

- **`social-ad`（内容电商/社媒广告）**：`9:16` 竖屏、**强 Hook** 前 3 格停滑点、默认无字幕、快节奏 KOC/生活流。= 现状抖音打法。
- **`listing-demo`（货架电商详情页/主图视频）**：`1:1`（或 16:9）、**弱 Hook**（前 3 秒仍抓人，但走「问题→产品价值」，不猎奇）、允许功能 callout 字幕、产品演示/真实使用/信任背书为主。

**九宫格 Hook→Solution→Trust 骨架两种 genre 都保留**，genre 只调 Hook 力度 / 比例 / 字幕策略 / 节奏。

**时长 / audio_mode 默认**（用户/全局可覆盖）：
- `social-ad`：9–15s（黄金，前 3 秒定生死）；audio 常 `ambient`/`music`。
- `listing-demo`：15–30s（讲清产品即可）；audio 常 `ambient`/`voiceover`（功能讲解口播）。

**默认映射（底层是「货架 vs 内容电商」）**：
- 货架电商（人找货：Temu/Noon/Amazon/Shein/Mercado/N11）→ 默认 `listing-demo`。
- 内容电商（货找人：默认档抖音/TikTok）→ 默认 `social-ad`。
- 站外引流/联盟投放（Temu 联盟投 TikTok、Amazon Inspire/Sponsored）→ 用户一句话切 `social-ad` 覆盖。

---

## 2. 通用红线（所有平台继承）

- 禁虚构功效、禁医疗/治疗承诺。
- 禁绝对化：最/第一/唯一/保证/100%/永久/立刻见效，**及各目标语言等价词**。
- 禁第三方品牌 logo / 水印 / 台标 —— **≠ 本产品自身 logo**（本产品 logo 按 `product-fidelity.md` 保留，这里禁的是别家品牌/平台水印）。
- 全部镜头遵守真实物理；不新增九宫格/脚本没有的镜头、场景、道具。

## 4. 平台档案

### 默认档（抖音/TikTok · 内容电商）
- 默认 genre：`social-ad`｜默认比例：`9:16`
- 语言/选角：用户指定，默认中/英 · 欧美女性
- 字幕：禁字幕
- 合规：仅通用红线
- 心智 · Hook：强 Hook 痛点驱动 · 停滑点（前 3 格）

### Temu（货架 · 全球极致低价）
- 默认 genre：`listing-demo`｜默认比例：**`1:1`**（可 3:4/16:9；**禁 9:16 入主图**）
- 语言/选角：英(US)为主/多语 · 泛欧美多元、亲民生活化
- 字幕：价值卖点 / 功能字幕可选
- 合规（叠加通用红线）：禁误导性紧迫感、禁虚假原价/折扣（EU DSA 监管收紧）
- 心智 · Hook：极致低价 + 超值 + 实用 + 即时满足；listing-demo 弱 Hook（超值/实用演示，不猎奇）
- ⚠️ **代码强约束**：主图视频只收 `1:1/3:4/16:9`，9:16 会被 X-Border 发布链路（`TEMU_VIDEO_ALLOWED_RATIOS`）拒绝并要求重出。

### Noon（货架 · 中东海湾 MENA）
- 默认 genre：`listing-demo`｜默认比例：`1:1`（可 16:9）
- 语言/选角：阿拉伯语(MSA) + 英 · 中东海湾形象 · 女性 modest 得体 · RTL 排版
- 字幕：阿/英双语字幕友好（功能点）
- 合规（叠加通用红线）：禁暴露、禁酒精/猪肉、禁宗教冒犯、禁 LGBT 暗示；斋月·开斋节可 seasonal 氛围
- 心智 · Hook：生活方式 + 身份 + 品质正品 + 家庭 + 送礼；中/弱 Hook，重场景代入与品质感

### Amazon（货架 · 全球最严 listing）
- 默认 genre：`listing-demo`｜默认比例：`1:1`（默认）或 `16:9`（A+/Sponsored Brands 常用）
- 语言/选角：分站点语言 · 中性专业选角
- 字幕：功能 callout 字幕鼓励
- 合规（叠加通用红线，最严）：禁价格/促销/折扣/倒计时文字 · 禁 badge（Amazon's Choice/Best Seller 仿冒）· 禁竞品对比贬低 · 禁站外引流 URL/“our website” · 禁伪造评价背书 · 禁时效词(sale/free shipping)
- 心智 · Hook：信任 + 参数规格 + 真实使用 + 评价心智 + 解决问题；弱 Hook（问题场景→产品核心功能→真实使用与信任背书）

### Shein（货架 · 快时尚，审美偏 social/试穿的例外）
- 默认 genre：`listing-demo`（审美偏 social/试穿）｜默认比例：`1:1`（试穿可切 9:16 竖版）
- 语言/选角：英为主/多语 · 年轻潮流多元
- 字幕：潮流/试穿 caption、功能字幕允许
- 合规（叠加通用红线）：美妆禁医疗功效；按市场禁不当暴露
- 心智 · Hook：潮流 + 穿搭上身 + 氛围感 + 平价 + 上新；节奏偏快、试穿/换装展示

### Mercado（Mercado Libre · 拉美货架）
- 默认 genre：`listing-demo`｜默认比例：`1:1`（可 16:9）
- 语言/选角：西语(墨/阿)/葡语(巴西) · 拉美形象
- 字幕：西/葡功能字幕允许
- 合规（叠加通用红线）：绝对化西/葡等价词（el mejor / garantizado 100%）；分期(cuotas)表述须真实合规
- 心智 · Hook：实用 + 家庭 + 性价比 + 可负担分期；中/弱 Hook

### N11（土耳其货架）
- 默认 genre：`listing-demo`｜默认比例：`1:1`（可 16:9）
- 语言/选角：土耳其语 · 土耳其/地中海形象 · 得体
- 字幕：土语功能字幕允许
- 合规（叠加通用红线）：土耳其广告法(Reklam Kurulu)；宗教文化尊重
- 心智 · Hook：性价比 + 家庭实用 + 正品信任；中/弱 Hook

> 约束来源：**仅 Temu 比例是代码强约束**（发布链路会拒）；其余平台的比例/字幕/合规为**运营建议（平台惯例）**。
