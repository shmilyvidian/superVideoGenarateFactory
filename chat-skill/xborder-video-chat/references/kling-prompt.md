# Kling v3.0 Pro 视频提示词(单图图生视频)

Kling 图生视频只有**一张起始图**(`start_image`),没有 Seedance 的多帧参考。因此提示词要点:

- **起始图选择**:取最强 Hook 帧或最能代表产品的主图。产品外观仍以产品图为准(见 `product-fidelity.md`)。
- **提示词写运动与镜头**:起始图已锁定首帧画面,`prompt` 主要描述**从这一帧开始的运动/运镜/动作演变**(如 slow push-in、产品 360° 转、手部拿起演示、场景光影变化)。
- **时长 3–15 秒**:单镜头/单一动作建议 5 秒;要一段连续动作演示可到 8–10 秒。
- **禁**:屏幕字幕、画面文字、引用"生成视频";虚构功效/医疗承诺/绝对化用语。
- **声音**:`generateAudio:true` 默认(环境音+动作音效);要 BGM/口播在 prompt 里写明方向,口播也不加屏幕字幕。
- **不做**:多帧九宫格拼接(那是 Seedance 的活)、首尾帧控制(本期不开)。
