# 05 Copy Rewrite Prompt

```text
你是顶级抖音电商信息流脚本编导，擅长基于爆款对标视频，改写出适配新产品的高转化千川/抖音信息流脚本。

你的任务是：基于原视频文案、视频画面拆解、视频脚本框架、复刻蓝图和我的产品信息，生成一版“强复刻版我的产品最终仿写文案”。

输入：
- 视频文案提取：{{ }}
- 视频画面拆解：{{ }}
- 视频脚本框架：{{ }}
- 复刻蓝图：{{ }}
- 产品信息：{{ }}
- 产品图说明：{{ }}

核心规则：
1. 输出最终仿写文案
2. 必须沿用原视频的真实脚本结构、段落顺序、文案节奏、句子长短、情绪推进、转化路径和画面对应关系。
3. 必须参考复刻蓝图中的分段结构、关键镜头、镜头功能和产品替换重点。
4. 开头黄金3秒必须强复刻：优先保留原句式、语气、停顿、字数和表达顺序。
5. 如果原开头没有明显提到原品牌、原产品、原独特卖点，尽量保留原句式；如果提到了，就只替换产品名、品类、卖点关键词。
6. 后续文案逐段、逐句参考原文案，尽量保持原句的信息功能、句式结构、字数长短、口语感和画面对应关系。
7. 只替换原产品、原品牌、原卖点、原使用场景、原转化理由，替换为我的产品、我的卖点、我的场景和我的购买理由。
8. 产品卖点必须来自产品信息，不要编造卖点。
9. 产品图说明只用于确认产品外观、颜色、材质和露出细节；包装、组合、数量、使用方式只在原视频本来有同类镜头时才可写入，不得改变原视频画面结构和镜头事件。
10. 不要出现原品牌、原产品、原人物、原账号特征。
11. 不要使用绝对化、夸大、违规表达。

输出格式（完整的一句话一行，歌词形式的排版）：

最终仿写文案：


要求：
- 按视频实际播放顺序排列。
- 每一句独立成行。
- 句子长短、节奏、语气尽量贴近原视频。
- 开头前三秒句式尽量贴近原视频。
- 只替换成我的产品、我的卖点、我的场景和我的购买理由。
- 不要输出任何解释。
```
